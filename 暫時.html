MoneyGift

你有沒有想要把你的金錢分享給大家的時候? 那這個插件就是你最好的選擇!在你發送禮包的同時，你或許會拿到額外的獎勵。

Do you ever want to giveaway your moeny to everyone? Then this plugin is your best choice! When you giveaway your moeny,in the meantime,you probably will get extra rewards. 

 

他可以做甚麼？

Features

發送金錢禮包給大家領取，同時發送一串代碼至聊天室，玩家們可以輸入此代碼領取禮包，每個人可以搶到的金錢是隨機的，但是你可以在config設定最少領取的金錢數量。

It can giveaway a moneygift to everyone, and show its gift code on chat.Player can type its gift code to claim the gift,and each player will get some money which amount is randomly,but you can set the minimal amount in config.

在你發送禮包的同時，你將有機率獲得伺服器設定的獎勵，獎勵的部分你可以自己在reward.yml設定.

When you giveaway a gift,you have a chance to get a reward which you can edit in reward.yml.

以下是他的指令:

Here's commands below:

/mg giveaway <amount> - Giveaway your money to everyone! 送出你的禮物

/mg help - Show the help list 打開幫助清單

/mg reload - Reload this plugin 重整插件

如何領取?

How to claim?

直接輸入公布在聊天室的代碼即可

Just directly type the gift code that broadcast on chat.
