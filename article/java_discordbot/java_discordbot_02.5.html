Day.2的下午 - JDA更新

12/19 2021

從0開始用Java編寫Discord 機器人!

Preface

痾...這裡是剛寫完Day 3的筆者，有人告訴我說最新的JDA的 GuildMessageReceivedEvent不能用了

所以中間插入一篇教大家怎麼改程式，我們 也是用舊的寫法歐，大家就自己改一下~

Start

其實只要改一點點點啦，首先我們先到pom.xml

把repositories刪掉，我們不需要他了!!

同時也把dependency的版本升上去

reload maven之後你會發現你的東西瞬間報錯超多，不要方!

我們到事件監聽的class，把 GuildMessageReceivedEvent 改成 MessageReceivedEvent

把上方import地方的有紅字那句整個刪掉

暴雷一下下次內容xd

這樣就可以喽
