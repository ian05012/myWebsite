12/19 2021

從0開始用Java編寫Discord 機器人!

Preface

機器人都擁有許許多多的指令，那究竟要怎麼把它做出來ㄋ?

Start

今天我講算是蠻暴力但簡單的方法，畢竟這裡是給新手看的為主xd

我自己所使用的是 這個影片 中所講解的方法，他在日後維護的話會比較方便閱讀，有興趣的可以試試看~

不過我們今天還是使用新手方法吧!

直接來看怎麼打，也是打在上次教學的監聽接收訊息裡面喔!

每個機器人都會有一個自己的前墜，否則一大堆的機器人放在一起，你同個指令可能會觸發好多個機器人xd

這裡我們一開始先宣告一個 prefix 的字串來當作他的前墜。

 

接著判斷接收到的訊息是不是指令，或者只是成員在聊天而已。

所以我們取收到的訊息內容，判斷他的開頭是不是你的前墜。

 

因為你可能會有很多個指令，所以宣告一個 command 字串代表你的指令內容，其實就是把指令跟前墜分開來

接著判斷你的指令內容是甚麼，這裡我隨便用 help 指令出來。

如果以上事情都對了，那機器人就會發送一段訊息給你。

喔對，我們機器人有頭像跟名字了xd

 

那如果你的指令有參數的話，我們在分離前墜跟指令內容的時候可以做點小變化

其實就是把你的字串切成一個陣列，切割點就是空格(\\s是空格的正則表達式)，例如你輸入"-say 我好帥 真的啦"

就會切成 [ say , 我好帥 , 真的啦 ] (因為我們有先把"-"替換掉了)

所以你索引值0的位置就會是指令內容，而後面的索引值就會是你的參數
