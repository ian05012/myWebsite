Day.1 - 讓機器人動起來!

11/14 2021

從0開始用Java編寫Discord 機器人!

Preface

Discord上的機器人百百種，但總是無法完全達到你的要求。既然這樣，不如自己寫一隻自己的機器人吧~

如果有任何說錯或者不清楚的地方，都可以告知我歐歐歐~

Prepare

  1. IDE(整合開發環境):  不專業的來說，就是寫程式的地方，有興趣的可以上網找詳細說明~ 這裡我所使用的是 IntelliJ IDEA，大家可以先下載下來~
  2. Java的基礎知識: 至少if else之類的基礎語法要先稍微了解一下，才不會都聽不懂喔~ 筆者自己也不是專業開發者，但是這樣寫下來發現簡單的Bot也不需要用到太複雜的知識啦w
  3. 一隻機器人: 在下面會說明~
  4. 一個測試用伺服器: 否則別人會把你當智障

那我們就開始囉~

Start

我們今天不會有程式的解釋，只會先讓大家把一些東西設定好，然後至少讓機器人上線~

首先我們先創建一個專案，我們今天是用Maven項目來做教學，所以左邊的地方記得要選取Maven~

當然如果你要用Gradle也是可以的，等等該複製的都會都會給.w.

接著輸入你的專案名稱就ok囉~

註:如果是第一次寫Java的話你需要安裝JDK，在最上面的Project SDK點下去之後會有Add SDK -> Download JDK，

接著選一個版本下載就好囉(不知道要選啥就跟我選一樣的吧XD)

創建好專案之後我們點到pom.xml

接著你會看到一托拉哭的東西不要害怕xd，我們沒有要一直用這些東西

我們到 JDA 的Github ，找到Download，把Maven下面那些通通複製起來~

JDA(Java Discord API) 就是我們這次寫機器人要用到的東東唷~

把dependency的

<version>VERSION<version> 

的地方改成我圈起來的那個版本(每個人去看的時候版本可能都不同了)

接著貼到pom.xml

接著在空白處右鍵
Maven -> Reload project

這樣他就會讀到我們的JDA囉

接下來我們要創建一個Class，作為我們的主程式~(名稱可以隨便打，但是你要知道你的這個Class是在做什麼的。未來的變數命名也是，養成良好習慣XD)

那麼IDE的部分先到這邊，接下來我們要創建一個機器人

我們到Discord的開發者網站

選擇New Application

起個名字

點到Bot -> Add Bot

OAuth2 把畫面中的東東打勾起來

複製連結貼到網址，就可以把機器人邀進你的伺服器囉!

接下來到Bot介面

把你的Token複製下來~

TOKEN絕對不要外流!!

TOKEN絕對不要外流!!

TOKEN絕對不要外流!!

很重要所以講三遍，因為有Token的人就可以隨意的搞你的機器人，如果你給予你的機器人權限很大，那他可以直接把你的伺服器摧毀掉OAO

以上都做完之後，你就把東西都設定完了~我們可以開始來編寫機器人了!!

我們一開始先直接把這個部分打上去手打一遍印象深刻加油，下一章開始才會開始解說~

基本上這裡我都是直接複製貼上，不會有太多的更改，所以這裡真的無法理解的話也沒關係，因為筆者自己也不是很理解，不過我會盡量說明

打完之後，我們點選右上角的Run

看到Finish Loading 的時候就OK囉

回到你的伺服器，你會發現你的機器人上線囉~ 恭喜你製作了你的第一個機器人!!
